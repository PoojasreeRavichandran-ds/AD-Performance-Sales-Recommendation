#  AD Performance Recommendation Engine

A web-based recommendation engine developed using **Python**, **Pandas**, and **Streamlit** to forecast  advertising performance using historical marketing KPIs.

##  Live Application

https://ad-performance-sales-recommendation-mwa9cbc973gfquf5npuosn.streamlit.app/

##  Features

- Forecast monthly advertising performance
- Predict monthly sales and units
- Uses historical KPIs for recommendations
- Interactive web interface
- Fast and easy to use

##  Forecasted Metrics

- Spend
- Clicks
- CPC
- CVR
- Ad Units
- ASP
- Ad Sales
- Ad ROAS
- ACOS
- Organic Sales
- Organic %
- Total Sales
- TASP
- Total Units
- Total ROAS
- TACOS

##  Technologies Used

- Python
- Streamlit
- Pandas
- OpenPyXL
- GitHub
- Streamlit Community Cloud

##  Application

The application allows users to:

- Enter Daily Spend
- Enter Number of Days
- Generate a monthly advertising performance forecast instantly
